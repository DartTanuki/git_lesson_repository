Небольшая памятка по работе с Git через консоль(cmd).
Здесь будут рассмотрены основные команды при работе с Git.


 L O G        C O M M A N D S
 ----------- Статус нашего репозитория ------------
 > git status
 --------------------------------------------------

 ----------- Информация о коммитах ----------------
 > git log
 --------------------------------------------------

------------ Информация о ветках в репозитории ----
> git branch
---------------------------------------------------


 О Б Щ И Е    К О М А Н Д Ы
------------ Создание репозитория -----------------
> git init
---------------------------------------------------

------------ Добавление файла к "отслеживающим" ---
> git add <название файла>
---------------------------------------------------

------------ Коммит файла/ов в репозитории --------
Общая форма:
> git commit -a -m "Комментарий"
"-a" - коммит для всех(all) файлов
"-m" - комментарий к коммиту

Для одного или нескольких файлов:
> git commit <название файла> ... <название файла> -m "Комментарий"
---------------------------------------------------

------------- Переключение между коммитами/ветками ---------
Переключение между коммитами:
    > git checkout <хэш коммита>
    Хэш коммита можно как полностью, так и первые 4-10 символов.
Переключение между ветками:
    > git checkout <имя ветки>

----------------------------------------------------

------------- Создание ветки -----------------------
> git branch <имя ветки>
----------------------------------------------------

------------- Удаление ветки -----------------------
> git branch -d <имя ветки>
"-d" - аргумент непосредственно для удаления ветки.
----------------------------------------------------



Р А Б О Т А     С   С Е Т Е В Ы М И     Р Е П О З И Т О Р И Я М И
------------- Добавление удаленного репозитория ----
> git remote add <название репозитория(любое вроде-бы)> <ссылка_на_репозиторий>
----------------------------------------------------

------------- Просмотр подключенных сетевых репов ----
> git remote -v
------------------------------------------------------

------------- Внесение изменений на сетевой реп -----
> git push -u <имя репозитория которые мы указывали> <ветка, из которой будут браться изменения>
-----------------------------------------------------

------------- Загрузка всех веток в сетевой реп -----
> git push <имя репозитория> --all
-----------------------------------------------------
